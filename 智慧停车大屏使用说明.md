# 智慧停车大屏使用说明

## 📋 项目概述

基于Vue 3 + ECharts 5.4 + Element Plus开发的智慧停车数据可视化大屏，提供实时停车场监控、数据分析和决策支持。

## 🎯 功能特性

### 核心功能模块

1. **实时监控模块**
   - 车辆进出实时统计
   - 车位利用率监控  
   - 设备状态监控
   - 告警信息提醒

2. **数据分析模块**
   - 收益分析统计
   - 支付方式分析
   - 停车时长分析
   - 车辆类型统计

3. **趋势预测模块**
   - 7天利用率趋势
   - 收入趋势分析
   - 智能预测建议

4. **区域管理模块**
   - 8个区域实时状态
   - 区域热力图
   - TOP区域排行

## 🚀 快速开始

### 访问方式

#### 方式一：首页快捷入口
1. 登录系统：http://localhost:8080/#/login
2. 进入首页后，点击"智慧停车大屏"快捷操作卡片

#### 方式二：直接访问
直接访问：http://localhost:8080/#/parking-dashboard

### 项目启动

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run serve

# 构建生产版本
npm run build
```

## ⚙️ 配置说明

### 数据更新频率

```javascript
// 时间更新：每秒
timeInterval: 1000ms

// 数据更新：每5秒  
dataInterval: 5000ms

// 图表刷新：实时
chartRefresh: immediate
```

### 性能配置

```javascript
// ECharts配置
renderer: 'webgl'        // WebGL渲染
progressive: 400         // 渐进式渲染
animation: true          // 启用动画
```

## 🚨 故障排除

### 常见问题

1. **图表不显示**
   - 检查ECharts版本兼容性
   - 确认DOM元素已正确挂载

2. **数据不更新**
   - 检查定时器是否正常运行
   - 确认数据服务接口正常

3. **页面卡顿**
   - 减少数据更新频率
   - 启用WebGL渲染模式

### 调试方法

```javascript
// 开启调试模式
console.log('数据更新:', realTimeData.timestamp)

// 检查图表实例
console.log('图表状态:', this.trafficChart)
```
