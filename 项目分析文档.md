# 雪人停车管理系统 - 项目分析文档

## 项目概述

### 基本信息
- **项目名称**: 雪人停车管理系统 (manage-front)
- **项目类型**: 企业级停车场管理后台系统
- **版本**: 0.1.0
- **开发状态**: 开发中
- **后端API地址**: http://www.xuerparking.cn:8543

### 项目定位
这是一个面向停车场运营管理的综合性SaaS平台，涵盖了从人员管理、车辆管理、设备维护到数据统计的完整业务流程。系统采用现代化的Web技术栈，提供直观易用的管理界面。

## 技术架构

### 前端技术栈
- **框架**: Vue 3.2.14 (使用Composition API)
- **UI组件库**: Element Plus 1.1.0-beta.1
- **路由管理**: Vue Router 4.0.11 (Hash模式)
- **状态管理**: Vuex 4.0.2
- **HTTP客户端**: Axios 0.21.4
- **构建工具**: Vue CLI 4.5.0
- **样式预处理**: Sass 1.26.5
- **语言**: JavaScript (ES6+)

### 特色功能库
- **二维码生成**: qrcode.vue 3.3.3
- **图片裁剪**: vue-cropperjs 5.0.0
- **富文本编辑**: wangeditor 4.7.15
- **Excel处理**: xlsx 0.16.0
- **图表展示**: vue-echarts 6.0.0, vue-schart 2.0.0
- **地区数据**: element-china-area-data 5.0.2
- **Cookie管理**: js-cookie 3.0.5
- **SVG图标**: svg-sprite-loader 6.0.11

## 系统架构特点

### 1. 权限控制系统
- **基于角色的访问控制(RBAC)**
  - 细粒度权限控制，每个功能模块都有独立的权限编码
  - 支持权限继承和权限组合
  - 动态菜单生成，根据用户权限显示对应菜单

### 2. 用户界面设计
- **响应式布局**
  - 支持桌面端和移动端访问
  - 自适应屏幕尺寸调整
- **现代化UI设计**
  - 基于Element Plus的组件化设计
  - 统一的视觉风格和交互体验
  - 支持主题切换和个性化配置

### 3. 数据管理
- **状态管理**
  - 使用Vuex进行全局状态管理
  - 标签页管理和页面缓存
  - 侧边栏折叠状态管理
- **数据持久化**
  - LocalStorage存储用户信息和权限
  - Token-based身份验证
  - 自动登录状态维护

### 4. 网络通信
- **HTTP请求封装**
  - 统一的请求拦截器和响应拦截器
  - 自动Token添加和错误处理
  - 请求超时和重试机制
- **API接口管理**
  - RESTful API设计
  - 统一的错误码处理
  - 接口版本管理
